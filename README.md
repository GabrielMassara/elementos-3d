# Elementos 3D

Repositório para armazenar sites com elementos 3D interativos criados usando Three.js, Spline e outras tecnologias de visualização 3D.

## Sobre o Projeto

Este repositório contém uma coleção de experimentos e projetos focados em elementos 3D para web, explorando diferentes tecnologias e bibliotecas para criar experiências visuais imersivas e interativas.

## Projetos Incluídos

### Macbook Interactive
Uma representação 3D interativa de um MacBook usando Spline Viewer.

**Características:**
- Modelo 3D interativo de MacBook
- Interface responsiva e moderna
- Efeitos visuais suaves com gradientes
- Tipografia elegante usando Inter font
- Interações com mouse

**Tecnologias utilizadas:**
- HTML5
- CSS3 com Flexbox/Grid
- Spline Viewer (@splinetool/viewer)
- Google Fonts (Inter)

## Tecnologias Utilizadas

- **Spline**: Ferramenta para criação e visualização de modelos 3D
- **Three.js**: Biblioteca JavaScript para gráficos 3D
- **HTML5**: Estruturação semântica
- **CSS3**: Estilização moderna com gradientes e animações
- **JavaScript**: Interatividade e funcionalidades dinâmicas
